Iris Species Prediction

A web applet to predict Iris species by metric attributes

Ronit Rudra
Budding Data Scientist

About the Applet

The Iris plant is a type of plant characterized by showy and colorful flowers. This applet was designed to help easily identify the species by relating to the size and shape of the petals and sepals. A very simple prediction model is the backend of this applet and requires the dimensions of the aforementioned petals and sepals. The applet is available Here

The Dataset

The model is trained using Anderson's Iris data.

The data has the following characteristics:

  • There are 4 numeric vairables encompassing the sepal and petal dimensions
  • The last variable is the species containing 3 classes namely `Setosa`, `Versicolor` and `Virginica`
  • Each class has 50 instances and will be the output variable
  • The numeric variables will act as the predictors

Model

The Random Forest algorithm is used as the prediction model and is perfect for such classification problems.

model <- train(Species~.,data=iris,method="rf")

Where would you use it?

This is a very rudimentary prediction application owing to the lack of large and diverse dataset on the same. A more complex and robust model can be built taking this app as a foundation.

So, If you are in a forest, come across a plant of the Iris family and you have the sudden urge to find out the correct species with no botanist in sight then don't worry. Just whip out your smartphone (provided you get a signal) and click here
